# Clothes-websites-
Sample project build website with PHP and HTML &amp; CSS
This project allows users to browse products, add them to cart, register/login, and includes an admin panel to manage products and orders.

Features

Browse and search products
Add products to cart
User login and registration
Admin panel for managing products and orders
Fully functional backend with PHP and MySQL
How to Run
Install XAMPP or WAMP (or any PHP + MySQL local server)
Open phpMyAdmin and create a new database, for example clothing_store
Import the file clothing_store into the new database
Open the website in your browser
